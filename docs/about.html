Why does this exist?

This website exists for my own personal use and from a desire to learn about SvelteKit and Github Actions. But I would be pleased if it turns out to be useful for others as well.

The function of this tool is to quickly search the freely available classic episodes from the Gizmoplex and to replace features that I miss from club-mst3k.com, like random episode selection and a daily "Watch It Together" episode.

I have no intention to monetize, take donations, or otherwise profit from this website in anyway. I will be happy to remove it when it is no longer needed or if I am asked to do so.

How does this work?

Data is collected from the Gizmoplex using a scheduled Github Action that runs a node.js script. This occurs once per day at around 9AM EST. It checks for newly available classic episodes and selects a new "Daily Experiment". This website displays the collected data.
